## Overview

This project is a **2D side-scrolling platformer** inspired by classic platforming mechanics.  
Built in Unity using C#, it explores **jump physics**, **moving platforms**,  
**hazard timing**, and **level layout design** across floating islands.

The player navigates obstacles, moving platforms, and hidden collectibles,  
testing timing skills, spatial awareness, and precise controls.
